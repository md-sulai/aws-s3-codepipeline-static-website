# AWS S3 Static Website Deployment using AWS CodePipeline

## Project Overview
A static website hosted on Amazon S3 with automated deployment using AWS CodePipeline.

## Technologies Used
- Amazon S3
- AWS CodePipeline
- AWS IAM
- HTML
- CSS

## Features
- Static website hosting
- Automated deployment
- Custom index page
- Custom error page

## Deployment Workflow
S3 (Source Bucket) → AWS CodePipeline → S3 (Deployment Bucket) → Static Website

## Learning Outcomes

- Learned how to host static websites using Amazon S3.
- Configured Static Website Hosting with custom index and error pages.
- Built an automated deployment pipeline using AWS CodePipeline.
- Understood the basics of CI/CD for cloud-based applications.
- Gained hands-on experience with AWS deployment workflows.

## Future Improvements

- Integrate GitHub as the source repository.
- Add AWS CodeBuild for automated build and testing.
- Deploy using AWS CloudFormation or Terraform.
- Use Amazon CloudFront for global content delivery.
- Configure HTTPS using AWS Certificate Manager.
